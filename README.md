# C-Project-Calculating-Bills
  A simple and efficient console-based application written in C++ to automate bill and invoice calculations.
## Author 
**Iman Hameed**
## 🚀 Features

* **Automated Calculations:** Computes total bills based on items, quantities, and electrcity prices dynamically.
* **Clean Logic:** Built using clean, readable C++ structures and basic programming fundamentals.
* **Interactive CLI:** Simple command-line interface making it user-friendly to navigate and test.

---

## 🛠️ Tech Stack & Concepts Used

* **Language:** C++
* **Concepts:** Variables, loops, conditional statements, and console input/output (`iostream`).
